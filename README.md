# Local-Content-Senegal

This project is an attempt to estimate the increased likelihood of having an internet connection, when more content is available in the local language.

# Senegal's Languages

 - French
 - Balanta-Ganja
 - Hassaniyya
 - Jola-Fonyi
 - Mandinka
 - Mandjak
 - Mankanya
 - Noon (Serer-Noon)
 - Pulaar
 - Serer
 - Soninke
 - Wolof
